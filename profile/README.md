## Fixing Terraform Cloud Setup Roadblocks

- Terraform Cloud login not working? Confirm organization access, identity provider settings, browser session state, and any required SSO permissions.  
- Workspace runs failing? Check provider credentials, Terraform Cloud variables, execution mode, run logs, and whether the connected repository path is correct.  
- API automation blocked? Rotate or verify the Terraform Cloud API token, confirm token scope, and compare requests against the terraform cloud api reference.  
- Private infrastructure unreachable? Review Terraform Cloud agents, network routing, firewall rules, and whether the agent pool is assigned to the right workspace.  
- Policy checks stopping applies? Inspect Terraform Cloud policy sets, review the failed policy output, and update configuration or approval rules before rerunning.
